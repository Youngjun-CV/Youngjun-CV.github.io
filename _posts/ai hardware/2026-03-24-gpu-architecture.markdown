## ⚙️ Processor & Programs
하드웨어 구조에 대해 알아가기전에 프로세서와 프로그램에 대해 먼저 알아가는게 좋다. 그럼 이게 뭔지 보자 + 그와 관련된 내용 확인하자

### 🟩 Processor란
명령어를 기반으로 유용한 작업을 수행하는 모듈. 명령어를 순차적으로 실행시키면서 우리가 원하는 알고리즘을 동작시킨다.  명령어의 주소값을 저장시키는 program counter 를 참조해서 실행시켜야 하는 명령어를 실행시킴. 명령어를 실행하면 레지스터, 메모리, CPU 상태 등과 같은 기계 상태가 변함
