## 🪀 순환신경망 (RNN)

### (1) RNN 의 개념과 핵심 아이디어
RNN 의 가장 혁신적인 점은 입력 시퀀스의 길이에 구애받지 않는다는 것입니다. 핵심 아이디어는 동일한 가중치 $W$ 를 매 단계마다 반복해서 적용 하는 것입니다. 이를 통해 모델은 이전 단계의 정보를 유지하며 시퀀스를 처리할 수 있게 됩니다.

### (2) RNN 의 구조와 수식
RNN 은 매 시점마다 은닉 상태 (hidden state) 를 업데이트하며 정보를 누적합니다. 구체적인 계산 과정은 다음과 같습니다.

* **단어 임베딩:** 단어 $x^{(t)}$ 를 벡터 $e^{(t)}$ 로 변환합니다.
$$e^{(t)} = E x^{(t)}$$

* **은닉 상태 업데이트:** 이전 시점의 정보 $h^{(t-1)}$ 와 현재 입력 $e^{(t)}$ 를 결합합니다.
$$h^{(t)} = \sigma(W_h h^{(t-1)} + W_e e^{(t)} + b_1)$$

* **출력 분포:** 현재 은닉 상태를 바탕으로 다음 단어를 예측합니다.
$$\hat{y}^{(t)} = \text{softmax}(U h^{(t)} + b_2)$$

### (3) RNN 의 장점과 단점
* **장점:**
  * 어떤 길이의 입력도 처리 가능합니다.
  * 이론적으로는 아주 먼 과거의 정보도 활용할 수 있습니다.
  * 입력이 길어져도 가중치가 공유되므로 모델 크기가 늘어나지 않습니다.

* **단점:**
  * 순차적 계산 특성상 병렬 처리가 어려워 연산 속도가 느립니다.
  * 실제로는 기울기 소실 문제 등으로 인해 아주 먼 과거의 정보를 유지하기 어렵습니다.

## ✏️ RNN 학습 하기

### (1) 학습 과정과 손실 함수
RNN 언어 모델은 전체 말뭉치를 입력받아 매 시점마다 다음 단어를 예측하도록 학습됩니다. 이때 각 단계의 손실은 실제 정답 단어와 예측 분포 사이의 **교차 엔트로피 (Cross-Entropy)** 를 사용합니다.

* **시점 $t$ 에서의 손실:**
$$J^{(t)}(\theta) = CE(y^{(t)}, \hat{y}^{(t)}) = -\sum_{w \in V} y_w^{(t)} \log \hat{y}_w^{(t)} = -\log \hat{y}_{x_{t+1}}^{(t)}$$

* **전체 평균 손실:**
$$J(\theta) = \frac{1}{T} \sum_{t=1}^{T} J^{(t)}(\theta) = \frac{1}{T} \sum_{t=1}^{T} -\log \hat{y}_{x_{t+1}}^{(t)}$$

### (2) Teacher forcing
학습 시에는 모델이 이전 단계에서 틀린 예측을 하더라도, 다음 단계의 입력으로는 실제 정답 단어 (**true word**) 를 넣어줍니다. 이를 **티처 포싱 (Teacher forcing)** 이라고 하며, 학습 초기 단계에서 모델이 빠르게 수렴하도록 돕습니다.

### (3) SGD 를 사용한 기울기 계산
전체 코퍼스에 대해 한꺼번에 기울기를 계산하는 것은 메모리상 불가능합니다. 따라서 **확률적 경사 하강법 (SGD)** 을 사용하여 문장 단위나 미니 배치 단위로 손실과 기울기를 계산하고 가중치를 업데이트합니다.

### (4) BPTT (Backpropagation Through Time)
RNN 의 기울기는 시간을 거슬러 올라가며 계산됩니다. 반복 사용되는 가중치 $W_h$ 에 대한 기울기는 각 시점에서 나타난 기울기들의 합으로 정의됩니다.

* **수식:**
$$\frac{\partial J^{(t)}}{\partial W_h} = \sum_{i=1}^{t} \frac{\partial J^{(t)}}{\partial W_h} \bigg|_{(i)}$$

* **예시:**
$t=4$ 에서의 손실에 대해 $W_h$ 의 기울기를 구할 때, $h^{(4)}, h^{(3)}, h^{(2)}, h^{(1)}$ 의 모든 시점을 거슬러 올라가며 미분값을 합산합니다. 실제 효율성을 위해 약 20 단계 정도만 거슬러 올라가는 Truncated BPTT 가 주로 쓰입니다.

RNN 을 이용한 텍스트 생성
RNN 은 학습된 확률 분포를 바탕으로 새로운 텍스트를 생성할 수 있습니다.
시작: 시작 기호 <s> 를 입력으로 넣습니다.
샘플링: 출력 분포 $\hat{y}^{(1)}$ 에서 단어를 샘플링 (sampling) 합니다.
반복: 샘플링된 단어를 다음 시점의 입력으로 다시 넣습니다.
종료: 종료 기호 </s> 가 나올 때까지 이 과정을 반복합니다.
이 방식을 통해 모델은 오바마의 연설문 스타일이나 해리포터 소설 같은 특정 문체를 흉내 낸 긴 글을 써 내려갈 수 있습니다.

---

언어 모델 평가 지표언어 모델의 성능을 측정하는 표준 지표는 펄플렉서티 (Perplexity, PPL) 입니다. 이는 모델이 다음에 올 단어에 대해 얼마나 헷갈려 하는지를 나타냅니다.

* **수식:**
* 
$$PPL = \prod_{t=1}^{T} \left( \frac{1}{P_{LM}(x^{(t+1)} | x^{(t)}, \dots, x^{(1)})} \right)^{1/T}$$

* **손실 함수와의 관계:**

$$PPL = \exp(J(\theta))$$

PPL 이 낮을수록 모델이 다음 단어를 더 확실하고 정확하게 예측한다는 의미이므로 좋은 모델로 평가받습니다. RNN 은 기존 n-gram 모델에 비해 PPL 을 획기적으로 낮추는 성과를 거두었습니다.
